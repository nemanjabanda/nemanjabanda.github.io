ServiceDesk Notifier

Application developed for ServiceDesk tickets notifications, it works using the provided ServiceDesk API. There is a support for multiple profiles, which can be changed quickly. Application places an icon in the system tray and from there displays popups for new tickets. Main window displays table with customizable information about arrived tickets. From there, ticket preview can be opened, or the application can open the ticket link in the selected browser. Additional options can be changed, like the type of the ticket to be shown, check time interval, notification sound, automatic startup.

Implemented using C#, WPF
